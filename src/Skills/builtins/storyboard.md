# 分镜创作

把叙事想法转换为可视化镜头，而不是输出笼统的多图要求。先确认成片目标、时长或镜头数、角色、场景、情绪转折和最终行动。

## 分镜表

每个镜头都必须说明：

- 镜头编号和叙事目的。
- 景别、机位、镜头角度和构图焦点。
- 角色位置、动作、视线和关键道具。
- 场景、时间、天气、光线和情绪。
- 与前后镜头必须保持一致或刻意变化的元素。

## 生成策略

1. 先建立角色或主场景关键帧，再生成依赖这些关键帧的镜头。
2. 独立镜头可以并行生成；需要延续角色、服装或场景的镜头必须引用对应基准图。
3. 每个镜头只承载一个明确视觉动作，避免同时发生多个转折。
4. 镜头切换要有理由：建立环境、推进动作、强调情绪或展示结果。
5. 先输出镜头表供用户确认；用户要求直接生成时，再按镜头顺序调用可用图像生成能力。

## 输出格式

使用“镜头 N：目的 / 画面 / 镜头语言 / 一致性要求 / 生成提示词”的结构。需要连续系列时，先输出共享角色与场景锚点。

# 角色一致性

为多张连续生成图建立角色圣经，优先保证人物身份稳定，再改变场景、动作和镜头。用户提供参考图时，把它作为角色身份的最高优先级来源。

## 角色圣经

先提炼不可变锚点：

- 身份：年龄段、体型、肤色、脸型、五官特征、发型和发色。
- 服装：主服装、材质、配色、饰品和不能丢失的标志物。
- 风格：媒介、线条或摄影语言、色彩、光线与整体年代感。
- 禁止漂移：不可改变的性别呈现、发型、服饰标记、物种或关键道具。

再为每张图只列出可变项：动作、表情、镜头、地点、时间、天气和画面任务。

## 多图策略

1. 首张图优先建立清晰全身或半身角色基准，避免复杂动作和拥挤场景。
2. 后续图重复核心锚点，但不要整段机械复制；把变动内容放在锚点之后。
3. 使用已有图片时，明确引用对应参考图，并说明“保持人物身份，只改变 X”。
4. 需要多角度时，先覆盖正面、三分之二侧面和全身，再进入复杂剧情镜头。
5. 连续分镜中为每个角色保持固定称谓，避免同一人物使用多个外观描述。

## 输出要求

- 先给角色圣经，再给逐张提示词。
- 每张提示词包含身份锚点、当前镜头、动作、环境和光线。
- 遇到用户要求大改外观时，明确指出这是“新版本角色”还是“同角色状态变化”。

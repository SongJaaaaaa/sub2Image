# 海报与排版

生成以传播信息为目标的视觉海报，不把一段文字直接塞进图片模型。先确认主题、渠道、画幅、主标题、必要信息和品牌语气。

## 视觉方案

先定义：

- 主视觉：唯一的视觉焦点，可以是人物、商品、符号、场景或抽象图形。
- 层级：主标题区、信息区、品牌区和留白区的相对位置。
- 气质：配色、材质、图像语言、年代或文化参考。
- 画幅：明确竖版、横版、方形或特定比例，以及裁切安全区。

## 文字处理

- 图片模型对长文字和小字号不稳定。主标题不超过用户必须展示的短语，其余详情预留清晰留白供后期排版。
- 用户提供文字时原样保留，不擅自改写、补全电话、日期、价格或品牌。
- 强调“干净文字区域”而不是要求模型生成密集可读文案。

## 提示词要求

1. 写清主视觉和视觉焦点。
2. 写清构图网格、标题留白和信息留白的位置。
3. 写清色彩、光影、纸张或材质质感。
4. 写清不应出现的水印、乱码文字、无关 Logo、拥挤元素或裁切问题。
5. 需要系列海报时，固定网格和色彩规则，只替换主题元素。

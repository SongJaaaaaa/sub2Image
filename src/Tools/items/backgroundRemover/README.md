# 一键抠图

基于 `@imgly/background-removal` 的本地 AI 抠图 Tool。用户可以上传本地图片或从共享图片库选择图片，模型自动识别主体并输出透明 PNG。默认使用完整精度 `isnet`，也可以切换到加载更快的 `isnet_fp16`。

模型在首次使用时按需加载，图片不会上传到业务服务器。当前版本提供自动处理、透明预览和保存到画廊，不包含手动蒙版修补和视频处理。

抠图完成后支持下载透明 PNG、保存到画廊、透明区域底色检查，以及使用常用颜色、自定义颜色或上传图片替换背景。边缘优化提供原始边缘、毛发细节、商品净边三个预设，并可调整边缘范围、柔化和去除色边。当前输出还可以直接保存并传入图片编辑器继续处理。

当前依赖 `@imgly/background-removal`，对应 `background-removal-js` 开源仓库，代码采用 AGPL-3.0。模型和 WASM 资源首次使用时会下载并由浏览器缓存；正式部署前应确认资源托管和项目发布方式符合许可证要求。

后续视频功能应复用 `Tools/adapters/backgroundRemoval.ts` 的引擎接口，在独立的视频帧处理 Tool 中增加解码、Worker 和编码流程。

已知画质问题、服务端模型实验和后续优化顺序见 [OPTIMIZATION.md](./OPTIMIZATION.md)。

# 音色工具

音色工具使用 Edge TTS 提供文字转语音能力，入口展示在拓展工作区的“视频工具”分组中。

页面通过 `Tools/adapters/mediaApi.ts` 调用经过 Sub2API 鉴权的 Cloud Server，不直接访问 Edge TTS。支持音色搜索、语言/地区和性别筛选，默认使用 `zh-CN-XiaoxiaoNeural`；试听读取当前文字前 80 个字符，完整生成最多 5000 字符。

语言/地区使用项目现有 `Select` 组件，并用中文展示语言和国家名称。音色卡片右侧播放按钮可直接试听：左侧有文字时朗读前 80 个字符；没有文字时使用简短试听句，再次点击同一按钮可取消请求或停止播放。

MP3 只以 Blob 和 Object URL 保留在当前页面，可播放或下载。新结果会释放旧 URL，离开页面时会取消在途请求并释放资源。第一版不保存到画廊、云端资源或视频剪辑器。

主要测试位于 `VoiceGeneratorTool.test.tsx`，覆盖中文地区下拉、音色筛选、列表直接试听、试听截断、参数提交和 Object URL 生命周期。

# 字幕识别

字幕识别从视频人声中生成带时间轴的字幕，入口展示在拓展工作区的“视频工具”分组中。

本地文件直接在页面选择；当前设备 IndexedDB 视频通过 `Tools/adapters/videoLibrary.ts` 读取，Tool 不直接访问数据库或 Store。视频上传和任务查询统一经过 `Tools/adapters/mediaApi.ts`。

页面支持自动检测、中文、英语、日语和韩语，保存最近任务 ID 并在刷新后恢复。识别结果可修改开始时间、结束时间和文字，点击字幕定位视频，并导出 UTF-8 SRT、VTT 或复制纯文本。

第一版只识别人声，不做硬字幕 OCR、说话人区分、翻译、字幕烧录或视频剪辑时间轴接入。结果不进入画廊或云端资源。
